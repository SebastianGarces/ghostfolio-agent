Ghostfolio AI Agent - Widget Dev Mode

How to use

This page lets you test the chat widget in isolation without running the full Ghostfolio app.

  1. Start the agent server: cd apps/agent && bun run src/main.ts
  2. Get a JWT from Ghostfolio (log in to Ghostfolio, open DevTools, run: sessionStorage.getItem('auth-token'))
  3. Paste the JWT below and click "Load Widget"
